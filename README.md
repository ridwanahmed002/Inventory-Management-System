• Admin login

• Search admin by username
• Edit employee information.

o Remove seller
o Update info
o Show Product info

o Sign in
o logout
o Show employee List
o Show seller list

o Search seller

• Show Product List

• Show Seller profile 
• Show customer profile 
• Update profile 
• Show All product 
• Login seller 
• Logout 
• Remove product
• Search product

• View Inventory
• Customer registration
• Search Inventory
• Delete Inventory
• View Cart
• Check out
• View order history
• Order confirmation
• Process payment
• Add to cart
• Remove from car
